## Overview
TreeBot AI is a command-line chatbot application written in Python that simulates a customer interaction and sales workflow for an eco-focused AI product. The program guides users through product education, purchasing decisions, pricing logic, and receipt generation.

This project was designed to demonstrate foundational programming concepts, user interaction flow, and business logic implementation.

---

## Features
- Interactive command-line chatbot experience
- Dynamic pricing based on quantity tiers
- Optional support plan logic
- Tax calculation and receipt generation
- Program restart functionality using recursion

---

## Technologies Used
- Python 3
- Command-line interface (CLI)

---

## Key Concepts Demonstrated
- Functions and modular logic
- Conditional statements
- User input handling
- Recursion for program flow control
- Business logic modeling

---

## Notes
This application runs locally in a controlled environment and is intended for educational and portfolio demonstration purposes.
